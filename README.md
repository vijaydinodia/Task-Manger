# 🚀 Task Manager (MERN Stack)

A full-stack **Task Management Web Application** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
This application allows users to efficiently manage daily tasks with secure authentication and role-based access.

---

## 📖 Overview

The Task Manager helps users organize their work by allowing them to create, update, and track tasks.  
It also includes an **Admin Panel** to manage all users and tasks, making it a complete real-world application.

---

## ✨ Features

- 🔐 Secure Authentication (JWT आधारित Login/Signup)
- 👤 Role-Based Access Control (User / Admin)
- ✅ Create, Read, Update, Delete (CRUD) Tasks
- 📋 Admin Dashboard (Manage Users & Tasks)
- 📊 Task Status Tracking (Pending / Completed)
- 🔎 Filter and manage tasks easily
- ⚡ RESTful APIs
- 📱 Responsive UI
